How to run this script


1.copy "php_crud_webtech2-main" folder

2.Paste inside root directory(for xampp xampp/htdocs, )

4. rename php_crud_webtech2-main to php-mysqli-crud

5.Open PHPMyAdmin (http://localhost/php-mysqli-crud)

6.Create a database with name mithila_db and admin_login

7.Import users.sql in mithila_db database file and test_crud.sql in admin_login database(given inside the zip package in SQL file folder)

6.Run the script http://localhost/php-mysqli-crud
